# List supported stemmer languages

List supported stemmer languages

## Usage

``` r
tnt_stemmers()
```

## Value

A character vector of language names accepted by the `stemmer` argument
of
[`tnt_text()`](https://strategicprojects.github.io/tantivyr/reference/tnt_field.md).

## Examples

``` r
tnt_stemmers()
#>  [1] "none"       "raw"        "portuguese" "english"    "spanish"   
#>  [6] "french"     "german"     "italian"    "dutch"      "russian"   
#> [11] "swedish"    "norwegian"  "danish"     "finnish"    "romanian"  
#> [16] "hungarian"  "turkish"    "arabic"     "greek"      "tamil"     
```
